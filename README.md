## Process

Clone the project:

```
git clone https://github.com/mrdoomy/hellocommunity.git
```

Install dependencies:

```
yarn install
```

Develop locally:

```
yarn start:dev
```

Run all unit tests:

```
yarn test
```

Compile the project:

```
yarn build
```
